# Hotel Room Booking System

## Project Description

The Hotel Room Booking System is a simple web-based application developed using HTML, CSS, and JavaScript. The project is designed to automate the hotel room reservation process by allowing users to enter customer details, select room types, choose check-in/check-out dates, and generate booking bills instantly.

This project helps in reducing manual work and demonstrates the implementation of basic hotel management functionalities in a user-friendly interface.

---

# Features

- Room Reservation System
- Customer Details Management
- Check-In and Check-Out Selection
- Automatic Bill Generation
- Different Room Types
- Simple and Responsive User Interface
- Real-Time Booking Confirmation

# How to Run the Project

1. Download or clone the repository

```bash

```

2. Open the project folder in VS Code

3. Open `index.html`

4. Run using:
- Live Server Extension
OR
- Double-click `index.html`

5. The project will open in your browser
